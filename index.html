Lesson Learning

Berikut merupakan rangkuman dari materi yang telah saya pelajari di Mata Kuliah Sistem Informasi Proses Industri.

Business Process

Proses bisnis adalah serangkaian langkah-langkah yang dilakukan untuk mencapai tujuan tertentu. Memahami alur kerja dan efisiensi proses bisnis sangat penting untuk meningkatkan produktivitas.

Data Flow Diagram (DFD)

DFD adalah alat grafis untuk menunjukkan bagaimana data mengalir dalam sistem. DFD membantu dalam memahami interaksi antara entitas, proses, dan penyimpanan data dalam sistem.

Entity Relationship Diagram (ERD)

ERD adalah representasi grafis dari entitas yang ada dalam sistem. ERD sangat penting dalam desain basis data.

HTML

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. HTML adalah dasar dari setiap situs web dan memungkinkan pengembang untuk mengatur konten dengan cara yang terstruktur.

CSS

CSS digunakan untuk mendesain dan memformat tampilan halaman web. Dengan CSS, pengembang dapat mengontrol tata letak, warna, font, dan berbagai elemen visual lainnya.
